hebcal-perl
===========

a perpetual Jewish Calendar (Perl)

by Danny Sadinoff

## DESCRIPTION
This is a partially-completed port of hebcal (originally written in C, circa 1994) to Perl.

We're preserving it here mostly for historical reasons.
